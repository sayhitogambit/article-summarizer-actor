# AI Article Summarizer

Summarize articles, documents, and long-form content using AI. Extract key points and create concise summaries.

## Pricing
- **Cost**: $0.005 per article
- **Charge**: $0.50 per article
- **Profit margin**: ~99%

## Features
- Multiple summary lengths (short/medium/long)
- Multiple formats (paragraph/bullets/abstract)
- Key point extraction
- Topic identification
- Compression ratio tracking

## Input
```json
{
  "text": "Long article text...",
  "length": "medium",
  "format": "paragraph",
  "includeKeyPoints": true,
  "model": "anthropic/claude-3.5-sonnet",
  "openrouterApiKey": "sk-or-v1-..."
}
```

## Use Cases
- Research paper summarization
- News article digests
- Content curation
- Meeting notes
- Document review
